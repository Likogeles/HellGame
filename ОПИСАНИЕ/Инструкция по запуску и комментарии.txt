Запуск через файл "main.py" PyCharm

Необходимы библиотеки:
	pygame
	time
	os
	sqlite3
	sys
	math
Файлы:
	data содержит карты уровней, музыку и спрайты
	classes.py - содержит классы
	dialogues.py - содержит функции диалогов
	functions.py - содержит функции
	main.py - содержит основной скрипт
	save.db - база данных сохранения
	scenes.py - содержит сцены игры
P.S.
	Видео тормозит из-за большой загруженности ПК, игра достаточно оптимизирована
